# <h1 id="stack-web">Les différentes stacks Web (MERN, MEAN, LAMP...)</h1>

| Base de données | Serveur web | Client (Frontend) | Serveur (Backend) | Stack |
| --------------- | ----------- | ----------------- | ----------------- | ----- |
| MongoDB         | Express     | React             | Node.js           | MERN  |
| MongoDB         | Express     | Angular           | Node.js           | MEAN  |
| MySQL           | Apache      | HTML/CSS/JS       | PHP               | LAMP  |
| MySQL           | NGINX       | HTML/CSS/JS       | PHP               | WEMP  |

<br/>

# <h1 id="intro-mern">Laboratoire d’introduction – La stack MERN</h1>

**Définitions :**

* **M = MongoDB** (Base de données NoSQL)
* **E = Express** (Cadriciel Node.js pour API REST)
* **R = React** (Composants frontend, SPA)
* **N = Node.js** (Backend JS, runtime)

<br/>

# <h1 id="historique-react">Historique de React</h1>

* Créé par Facebook en 2013.
* Révolution avec **JSX** et le concept de **Virtual DOM**.
* Grande mise à jour en 2019 avec les **Hooks**.
* **SPA** : Single Page Applications.
* React est une **bibliothèque** (et non un framework).

<br/>

# <h1 id="etape1-node-install">Étape 1 – Installer Node.js et créer l'application React</h1>

## Objectif :

Lancer un projet React de base.

### Instructions :

```bash
node -v          # Vérifier la version de Node
npm -v           # Vérifier npm (Node Package Manager)

cd C:\projetsreact
npm install -g create-react-app
npx create-react-app demo1
cd demo1
npm start
```

**Résultat attendu :**
Le navigateur s’ouvre à `http://localhost:3000` avec l'application React par défaut.

# <h1 id="etape6-composants">Étape 6 – Comprendre les composants React</h1>

Deux types de composants :

* Composants **fonctionnels**
* Composants **de classe**

**Exemple de composant fonctionnel :**

```jsx
const App = () => {
  return <h1>Bonjour !</h1>
}
```

<br/>

# <h1 id="etape7-fonction-classe">Étape 7 – Transformer un composant fonction en classe</h1>

```jsx
import { Component } from 'react'

class App extends Component {
  render() {
    return <h1>Bonjour !</h1>
  }
}
```

<br/>

# <h1 id="etape8-stateless">Étape 8 – Ajouter des composants stateless</h1>

```jsx
const Ordinateur = () => <div>Je suis un ordinateur</div>
```

Inséré dans `App` :

```jsx
<Ordinateur />
<Ordinateur />
```

# <h1 id="etape9-props">Étape 9 – Props simples</h1>

### Exemple :

```jsx
const College = ({ categorie, endroit }) => (
  <div>Type : {categorie}, Ville : {endroit}</div>
)

<College categorie="public" endroit="Montréal" />
```

<br/>

# <h1 id="etape10-props-children">Étape 10 – Props avec children</h1>

```jsx
<College categorie="privé" endroit="Québec">
  Fermé les weekends
</College>
```

<br/>

# <h1 id="etape11-props-alternatif">Étape 11 – Accès aux props via `props.nom`</h1>

```jsx
const College = (props) => (

    {props.categorie} – {props.endroit}
    {props.children}
  </div>
)
```

# <h1 id="etape12-state-intro">Étape 12 – Ajouter un state</h1>

```jsx
class App extends Component {
  state = { cat: "public", lieu: "Gaspésie" }

  render() {
    return <College categorie={this.state.cat} endroit={this.state.lieu} />
  }
}
```

<br/>

# <h1 id="etape13-setstate">Étape 13 – Modifier le state avec un bouton</h1>

```jsx
changerCollege = () => {
  this.setState({ cat: "privé" })
}

<button onClick={this.changerCollege}>Changer</button>
```
