<h1 id="stack-web">Les différentes stacks Web (MERN, MEAN, LAMP...)</h1>
<ul>
  <li><strong>MERN</strong> = MongoDB + Express + React + Node.js</li>
  <li><strong>MEAN</strong> = MongoDB + Express + Angular + Node.js</li>
  <li><strong>LAMP</strong> = Linux + Apache + MySQL + PHP</li>
  <li><strong>MAMP</strong> = Mac + Apache + MySQL + PHP</li>
  <li><strong>WAMP</strong> = Windows + Apache + MySQL + PHP</li>
  <li><strong>WEMP</strong> = Windows + Nginx + MySQL + PHP</li>
</ul>

<h1 id="intro-mern">Laboratoire d’introduction – La stack MERN</h1>
<p>La stack MERN est composée de :</p>
<ul>
  <li><strong>M</strong> = MongoDB (base de données NoSQL)</li>
  <li><strong>E</strong> = Express (framework backend)</li>
  <li><strong>R</strong> = React (vue – frontend)</li>
  <li><strong>N</strong> = Node.js (environnement d’exécution JavaScript côté serveur)</li>
</ul>

<h1 id="historique-react">Historique de React et concepts clés (SPA, JSX...)</h1>
<ul>
  <li>React est développé par Facebook (2013)</li>
  <li>Mise à jour majeure en 2019</li>
  <li>React permet de créer des applications de type SPA (Single Page Application)</li>
  <li>JSX = JavaScript + XML pour écrire des composants</li>
</ul>
